# Menu

This is a C++ version of Quacker's program meant to help me practice my usage of Git while collaborating with others

## Features

- Prints a menu to the console and lets the user add or multiply 2 numbers
- Verifies that user input is a valid number
- Saves results to a .txt file

## Requirements

- You need to be able to run compiled C++ programs on your computer

## How to run

- Navigate to the directory on your computer with the compiled C++ program, then in Linux use ./Menu to run it
