# MenuDriven 
Runs the user through simple math drills to generate data to store, retrieve, and analyze.

## How to run
- requires Python 3
- download main.py, menu.py, login.py, and drill.py
- on the command line type: python3 main.py

### Objective
I'm creating a menu driven console program. The point of which is academic & social, not commercial or even necessarily practical. I hope that the tools and techniques developed here are stolen and put to good use. These particular menus are developed around the idea of quizzing oneself with math challenges, recording, retrieving, and analyzing the data.
