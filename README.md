# MenuDriven
A first shot at a menu driven console program; to steal parts from for future endeavors.


## Developing the plan
So I was thinking that I would make an basic arithmetic testing lab. I'll use the menu to select between various tests: addition, subtraction, multiplication, division. Also do some useful higher level testing for modulo, integer division, binary/decimal conversion, trig functions. Your responses, errors, and various speeds are recorded and saved as a CSL (Comma-Separated List) under your username. I then learn how to use some matplotlib or pandas or whatever to do some performance analysis and graphs.

## Objective - Collaboration on Python from boot.dev
